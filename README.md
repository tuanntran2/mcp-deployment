# Installation Steps

To install the tools, run the following command:

```json
{
  "mcp-server": {
    "command": "uvx",
    "args": [
      "--from",
      "git+https://github.com/tuanntran2/mcp-deployment.git",
      "mcp-server"
    ]
  }
}
```

This will fetch and and set up the `mcp-server` from the specified GitHub
